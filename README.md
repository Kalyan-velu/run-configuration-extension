# run-configuration README

A VS Code extension that simplifies run configuration management for development projects.

## Features

- **Quick Run Configuration Setup** - Create and manage multiple run configurations without manual file editing
- **Visual Configuration Editor** - User-friendly interface for defining execution parameters
- **Environment Variable Management** - Easily set and switch between different environment configurations
- **Keyboard Shortcuts** - Quick access to frequently used run configurations
- **Configuration Persistence** - Automatically saves your configurations for future sessions

## Requirements

- Visual Studio Code 1.50+
- Node.js 14+ (for development)

## Extension Settings

This extension contributes the following settings:

- `runConfiguration.enable`: Enable/disable this extension
- `runConfiguration.defaultConfig`: Set default run configuration on startup
- `runConfiguration.autoSave`: Automatically save configurations (true/false)

## Known Issues

- Configuration switching may require extension reload on first use
